OPTIX Prototype - 2D Simulation Demo
Files:
- index.html : main web page. Open in browser.
- style.css : styles.
- app.js : simulation logic (A* pathfinding, agent movement, KPI).
- layout.json : sample layouts (before/after).
How to run:
1. Unzip optix_prototype.zip
2. Open index.html in your web browser (Chrome, Edge recommended).
3. Use controls (scenario, load, start, pause, speed) to run demo.
Notes:
- This is a simplified demo for presentation purpose (grid-based, simplified KPIs).
- If you want, I can extend: multiple agents, continuous tasks, export results, Electron packaging(.exe), or upgrade to 3D (Three.js).
